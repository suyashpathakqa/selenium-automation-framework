# Selenium Automation Framework (SauceDemo)

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports

## Features
- Page Object Model (POM)
- Data-driven testing (Excel)
- Screenshot on failure
- Logging (Log4j)
- TestNG Listeners
- CI/CD ready (Jenkins)

## Test Scenarios
- Login Test
- Checkout Flow (E2E)

## How to Run
```bash
mvn clean test
